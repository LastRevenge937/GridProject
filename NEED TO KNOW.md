# These are things that YOU NEED TO KNOW

* GFID, stands for **Global File Identification**, every file gets a GFID whether from the **Dockyard** cell ({user} created files get GFID here) or a **Gate** cell (go see CellTypes.md for more infor on my cell names and their purposes)
* **SENTINEL** this is an **AI**, SENTINEL oversees your grid, and sends you pings.
*  **Shadow Grid** IS ALSO AN **AI**, Shadow Grid uses copies of your grid, and firewall (Ill include one if you dont have a firewall) to test for vulnerabilities. Shadow Grid sends **YOU** pings **and** pings SENTINEL 
* **LAYERS** there are 12 layers to this grid,
       Layer 0: **Limbo**
       Layer 1: **Courtyard**
       Layer 2: **Infastructure**
       Layer 3: **Operations**
       Layer 4: **Escort Zone**
       Layer 5: **Inter Castle Exchange**   #I origanally made this for me AND some of my friends to use, so yes you can connect with your friends
       Layer 6: **Quaratine City**
       Layer 7: **PURGATORY**
       Layer 8: **CROWN VAULTS**
       Layer 9: **SPECIAL AGENT FACTORY**  #Go watch TRON Legacy, these agents are similar to Rinzler
       Layer 10: **PSG (Private Shadow Grid)**
       Layer 11: **Dead-Man-Switch + GSG (Global Shadow Grid)**  #Yep, I made a Global version which has a news feed if you have multiple friends with this Grid.
